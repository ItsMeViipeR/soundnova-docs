# SoundNova installation

## Download the installer

To download SoundNova, go on the project's [Github](https://github.com/ItsMeViipeR/soundnova) to download the latest version.

## Install SoundNova

To install SoundNova, nothing could be simpler. Once you have [downloaded the installer](#download-the-installer), launch it and the installation will be done automatically.

## Launch SoundNova

When you launch SoundNova, the software will ask you for the default path to your local media folder. This is the only configuration you will have to do for the program to work correctly.

Other settings are available, notably for [Discord presence](https://discord.com/developers/docs/rich-presence/best-practices) or for other more specific settings.
