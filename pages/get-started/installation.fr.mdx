# Installation de SoundNova

## Télécharger le programme d'installation

Afin de télécharger SoundNova, rendez-vous sur le [GitHub](https://github.com/ItsMeViipeR/soundnova) du projet afin de télécharger la dernière version.

## Installer SoundNova

Pour installer SoundNova, rien de plus simple. Une fois que vous avez [télécharrgé le programme d'installation](#télécharger-le-programme-dinstallation), lancez-le et l'installation se fait automatiquement.

## Lancer SoundNova

Au lancement de SoundNova, le logiciel vous demandera le chemin par défaut vers le dossier de vos médias locaux. C'est la seule configuration que vous aurez à faire pour que le programme fonctionne correctement.

D'autres réglages sont disponibles, notamment pour la [présence Discord](https://support.discord.com/hc/fr/articles/115001557452-Invitation-aux-jeux-et-statut-d%C3%A9taill%C3%A9-Rich-Presence-) ou pour d'autres paramétrages plus pointus.
