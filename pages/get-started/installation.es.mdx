# Instalación de SoundNova

## Descarga el instalador

Para descargar SoundNova, vaya al Github del proyecto para descargar la última versión.

## Instalar SoundNova

Para instalar SoundNova, nada podría ser más sencillo. Una vez que haya [descargado el instalador](#descarga-el-instalador), ejecútelo y la instalación se realizará automáticamente.

## Inicie SoundNova

Cuando inicie SoundNova, el software le pedirá la ruta predeterminada a su carpeta de medios local. Esta es la única configuración que tendrás que hacer para que el programa funcione correctamente.

Hay otras configuraciones disponibles, en particular para [Presencia de Discord](https://discord.com/developers/docs/rich-presence/best-practices) o para otras configuraciones más específicas.
